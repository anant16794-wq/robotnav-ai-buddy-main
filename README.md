# RobotNav Assist Dashboard

RobotNav Assist is a responsive command center for an assistive robot car. It brings voice control, route guidance, ESP32 command simulation, emergency stop controls, and family monitoring into a single production-ready dashboard.

## Local Development

```bash
npm install --legacy-peer-deps
npm run dev
```

The development server is configured for `http://127.0.0.1:8080/`.
